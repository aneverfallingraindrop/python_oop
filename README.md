# Балльно-рейтинговая система (БРС)

Максимальное количество баллов за курс — **200**.

Распределение баллов:

* 4 теста — 40 баллов
* 7 лабораторных работ — 70 баллов
* 2 контрольные работы — 40 баллов
* Экзамен — 50 баллов

Итоговая оценка формируется на основании суммарного количества набранных баллов.
